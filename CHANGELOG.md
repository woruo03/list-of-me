# 更新日志

本文件用于记录项目的重要变更。

## v0.2.0 - 2026-03-19

### 新增
- 使用 Tailwind CSS + daisyUI 对布局、卡片、面板和弹窗进行了毛玻璃风格视觉升级。
- 为按钮和卡片增加了更顺滑的悬浮动效与过渡效果。
- 新增仓库级 `CHANGELOG.md`，并在设置页直接读取该文件展示更新日志。

### 变更
- 将前端、Tauri、Rust 元数据和 PKGBUILD 的版本统一升级到 `0.2.0`。
- 将筛选与选择操作统一到单卡片交互结构中，界面更一致。
- 优化对齐规则：
  - 筛选内容左对齐。
  - 选择操作按钮右对齐。
- 优化滚动容器结构，内容区独立滚动，标题栏保持固定。
- 截止时间选择器从内嵌面板改为弹窗交互。
- 优化下拉菜单可读性与交互：
  - 修复下拉被卡片遮挡的问题。
  - 下拉宽度可按内容扩展，避免长文案被截断。
  - 增强筛选相关按钮的边框对比度。
- 优化弹窗视觉风格：
  - 降低背景遮罩强度。
  - 弱化外圈阴影，减少“黑圈”观感。

### 修复
- 修复项目详情页打开筛选后布局错位、换行异常的问题。
- 修复设置页下拉菜单被容器裁切、显示不完整的问题。
- 修复项目详情页“编辑项目”弹窗遮罩过暗、外圈阴影过重的问题。
- 修复截止时间弹窗遮罩过暗导致界面压黑的问题。
- 修复任务到期后状态提示缺失的问题：现在会显示“已逾期”。

## v0.1.0 - 2026-03-14

### 新增
- 基于 Tauri + Vue 3 的桌面版首个版本发布。
- 提供任务与项目管理核心流程（增删改查）。
- 提供主题与字体大小设置。
- 集成本地 SQLite 数据持久化与自动更新能力。
